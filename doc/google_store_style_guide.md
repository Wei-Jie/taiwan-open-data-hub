# Google Store 風格轉型指南 (Visual Identity Guide)

## 1. 色彩規範 (Color Palette)
不再使用硬性漸層，改用 Google 特有的中性背景色與重點色：
- **背景 (Surface)**: `#F8F9FA` (淺色) / `#1F1F1F` (深色)
- **卡片 (Container)**: `#FFFFFF` (淺色) / `#2D2E31` (深色)
- **重點色 (Primary)**: `#1A73E8` (Google Blue)
- **文字 (Text)**: `#202124` (主要) / `#5F6368` (次要)

## 2. 圓角與陰影 (Radius & Shadows)
- **大卡片**: `28px` (符合 Material 3 規範)
- **按鈕與標籤**: `100px` (藥丸形狀)
- **陰影**: 移除強烈投影，改用極細的 Border (1px) 或非常柔和的擴散陰影。

## 3. 字體排版 (Typography)
- **標題**: 增加 `letter-spacing: -0.02em`，使用更粗的字重。
- **正文**: 增加 `line-height: 1.6` 提升閱讀性。

## 4. 元件優化方向
- **Tab 切換**: 改為底部分割線或是帶有背景色的藥丸狀切換器。
- **資料卡片**: 移除卡片內部的漸層背景，改用純色背景搭配極簡的 Icon。
- **地圖**: 優化控制介面，使其懸浮於地圖上方且具備更大的圓角。
