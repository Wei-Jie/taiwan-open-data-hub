# 人機協作檢查清單

*此檢查清單用於確保 BA Agent 與人類用戶的協作過程品質*  
*適用於所有需要人類確認的關鍵決策點*

---

## 協作前準備檢查

### BA Agent 準備事項
- [ ] **資訊整理完整**：所有相關分析結果已整理為易理解的格式
- [ ] **選項明確**：為人類提供了清楚的選項和每個選項的後果說明
- [ ] **背景充分**：準備了充足的背景資訊和分析依據
- [ ] **視覺輔助**：準備了圖表、清單等視覺化工具協助理解
- [ ] **語言簡潔**：使用非技術性語言，確保人類用戶可以理解
- [ ] **時限合理**：設定了合理的確認時限，避免流程延滯

### 人類用戶準備確認
- [ ] **時間安排**：人類確認者有足夠時間進行充分的審查
- [ ] **權限確認**：確認者具有相應的決策權限
- [ ] **背景了解**：確認者了解專案背景和當前進度
- [ ] **相關資料**：確認者可以存取所有相關的背景資料

## 檢查清單使用說明

### 使用時機
- **協作前**：使用「協作前準備檢查」確保準備充分
- **協作中**：使用各階段檢查確保協作品質
- **協作後**：使用「後續行動檢查」和「品質改進檢查」確保效果

### 責任分配
- **BA Agent**：負責 AI 相關的檢查項目
- **人類確認者**：負責人類相關的檢查項目
- **專案管理者**：負責整體協作品質監督

### 品質標準
- **必須項目**：標記為 [ ] 的項目必須完成
- **檢查頻率**：每個協作點都需要使用相應的檢查清單
- **記錄要求**：檢查結果需要記錄在協作記錄中

### 持續改進
- 根據使用經驗定期更新檢查清單
- 收集用戶反饋持續優化協作流程
- 建立檢查清單效果的評估機制
