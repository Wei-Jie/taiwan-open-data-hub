# 3. 用戶故事與介面設計 (User Story and Design)

## 何時使用

當您已經有了一份清晰的需求文件（如 FRD），並需要將其轉化為開發團隊可以執行的用戶故事、任務，並搭配初步的 UI/UX 設計建議時，請使用此 workflow。它旨在彌合產品與開發之間的差距，確保功能設計貼近用戶需求。

## 快速模板

```
作為一個跨功能團隊（包含 PM、Developer、QA），請根據我提供的 FRD 文件，完成以下任務。

**FRD 文件片段:**
---
[此處貼上您的 FRD 文件，或其中一個功能模組的詳細描述]
---

**任務:**
1.  **拆分史詩故事 (Epic):** 將大的功能模組拆分為更小的、可管理的用戶故事 (User Stories)。
2.  **定義技術任務 (Technical Tasks):** 為每個用戶故事列出實現它所需的後端和前端開發任務。
3.  **設計 UI/UX 建議:** 為每個用戶故事提供線框圖（Wireframe）的文字描述或 Markdown/Mermaid 結構，描述頁面佈局、關鍵元件和用戶互動流程。
4.  **撰寫測試案例 (Test Cases):** 為每個用戶故事提供主要的使用者驗收測試（UAT）案例。
```

## 完整範例

```
作為一個跨功能團隊（包含 PM、Developer、QA），請根據我提供的 FRD 文件，完成以下任務。

**FRD 文件片段:**
---
**模組: 儀表板 (Dashboard)**

- **用戶故事:** 作為 HR 主管，我希望在儀表板上看到即時的面試統計數據，以便快速了解招聘進度。
- **需求:**
  - 顯示"進行中的面試"、"今日已完成"、"本週總覽"三個關鍵指標。
  - 以圖表形式展示近 7 天的面試趨勢。
  - 列表顯示今日待辦面試。
- **非功能性需求:** 儀表板數據應每 5 分鐘自動刷新。
---

**任務:**
1.  **拆分史詩故事:** 將儀表板功能拆分為更小的用戶故事。
2.  **定義技術任務:** 為每個故事列出開發任務。
3.  **設計 UI/UX 建議:** 提供儀表板的佈局描述。
4.  **撰寫測試案例:** 提供主要的 UAT 案例。
```

## 產出說明

您將會得到一份詳細的開發執行計劃，包含以下部分：

-   **1. 用戶故事拆解 (User Stories):**
    -   **Story 1:** 作為 HR 主管，我能在儀表板頂部看到三個核心 KPI（進行中、今日完成、本週總覽），以便一眼掌握關鍵數據。
    -   **Story 2:** 作為 HR 主管，我能看到一個折線圖，展示過去 7 天的面試數量趨勢，以便分析招聘效率。
    -   **Story 3:** 作為 HR 主管，我能在儀表板上看到一個"今日待辦"列表，以便安排工作。

-   **2. 技術任務 (Technical Tasks):**
    -   **後端:**
        -   [Task-BE-1] 創建一個 API 端點 (`/api/dashboard/kpi`) 提供核心 KPI 數據。
        -   [Task-BE-2] 創建一個 API 端點 (`/api/dashboard/trends`) 提供趨勢圖數據。
        -   [Task-BE-3] 優化數據庫查詢，確保 API 在 500ms 內響應。
    -   **前端:**
        -   [Task-FE-1] 開發 KPI 卡片元件。
        -   [Task-FE-2] 使用 Chart.js 或類似庫整合趨勢圖。
        -   [Task-FE-3] 實現數據自動刷新邏輯。

-   **3. UI/UX 設計建議 (Wireframe Description):**
    -   **佈局:** 三欄式佈局。
    -   **頂部:** 三個並排的卡片（Card Component），分別顯示 KPI 數據。
    -   **左側:** 折線圖區塊，佔據約 2/3 寬度。
    -   **右側:** "今日待辦"列表，佔據約 1/3 寬度。

-   **4. 使用者驗收測試案例 (UAT Cases):**
    -   **TC-1:** 驗證 KPI 數據是否與後台數據庫的即時統計一致。
    -   **TC-2:** 驗證趨勢圖是否正確顯示了過去 7 天的數據。
    -   **TC-3:** 驗證頁面是否在無手動刷新的情況下，約每 5 分鐘更新一次數據。

這份產出為 Sprint Planning 提供了極佳的輸入，讓團隊能快速投入開發。
